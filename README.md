## 💾 Instalação

Clone o repositório e vá até o diretório do projeto:
```
git clone https://github.com/SadS4ndWiCh/minesweeper-nextjs.git && cd minesweeper-nextjs
```

Instale todos os pacotes:
```
yarn install
```

E já pode inicializar:
```
yarn dev
```
